# runcom

- vimrc for vim
- zplugrc for zplug

## vimrc

```bash
# install vundle
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
# launch vim and run :PluginInstall
```

## zplugrc

```bash
# install and zplug
git clone git@github.com:zplug/zplug.git ~/
cat zplugrc >> ~/.zshrc
# install zsh plugins
zplug install
```
